## 	:hammer_and_wrench: Built Using:
- Android Studio - develop in Java
- Room Database (Local DB).
- MVVM (Model-View-ViewModel) - android structured.
  - Android View Model
  - Repository
  - LiveData
- Navigation - NavGraph, NavController.
- Build HTML popup viewer.
